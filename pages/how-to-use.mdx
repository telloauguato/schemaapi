# How to Use

Using SchemaAPI is straightforward:

1. Create a public repository on Github.

2. Set `main` as your default branch.

3. Create the schema file: `*.schema.json`.

    ``` json showLineNumbers copy filename="*.schema.json"
    {
        "name": "schemaName", // Schema name
        "description": "This is the schema description", // Schema description
        "length": 1, // Amount of data returned. Default: 1; -x: randon between 1 and x
        "content": [] // Enter the data types here
    }
    ```

| key | type | description |
|---|----|-----|
| name | ```string``` | Schema Name |

4. Now just request the schema with the url: `https://schemaapi.vercel.app/api/<repository>@<username>/<pathAndNameOfSchemaFile>`. No need to append `.schema.json`
