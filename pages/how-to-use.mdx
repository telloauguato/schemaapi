# How to Use

Using SchemaAPI is straightforward:

1. Create a public repository on Github.

2. Set `main` as your default branch.

3. Create the schema file: `*.schema.json`.

    ``` json showLineNumbers copy filename="*.schema.json"
    {
        "name": "schemaName",
        "description": "This is the schema description",
        "length": 1,
        "content": [] 
    }
    ```

	| key | type | description | default |
	|---|----|-----|----|
	| name | ```String``` | Schema Name. | ```"schemaapi"``` |
	| description | ```String``` | Schema description. | ```"My SchemaAPI"``` |
	| length | ```Number``` | Amount of data returned. -x â€” randon between 1 and x. | ```1``` |
	| content | ```Array<Types>``` | Array containing the bodies with types of the data to be inserted. | ```[]``` |

4. Now just request the schema with the url: `https://schemaapi.vercel.app/api/<repository>@<username>/<pathAndNameOfSchemaFile>`. No need to append `.schema.json`
