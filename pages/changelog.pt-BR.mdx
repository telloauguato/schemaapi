# Changelog

## Versão 1.0.0 (30 de Junho de 2024)

- **Primeira Versão:**
  - Lançamento inicial do SchemaAPI, uma ferramenta para geração de dados fictícios personalizáveis para protótipos de sistemas.

- **Novos Tipos Adicionados:**
  - **`name`**: Adicionado suporte para geração de nomes fictícios.
  - **`number`**: Introduzido para geração de números, com suporte a intervalos personalizáveis.
  - **`email`**: Incluído para geração de endereços de e-mail fictícios.
  - **`username`**: Implementado para geração de nomes de usuário simulados.
  - **`pattern`**: Adicionado para definir padrões personalizados de geração de dados.
  - **`options`**: Introduzido para configuração de opções predefinidas para escolha aleatória.
  - **`schema`**: Incluído para definição de esquemas aninhados personalizados.
  - **`uuid`**: Implementado para geração de identificadores únicos universais.
  - **`address`**: Adicionado suporte para geração de endereços fictícios completos.

Esta versão inaugural do SchemaAPI marca o início do desenvolvimento de uma ferramenta robusta e versátil para geração de dados fictícios, visando facilitar o processo de prototipagem e desenvolvimento de sistemas.
