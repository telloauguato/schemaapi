# Usage Examples

Here are some examples of how you can use SchemaAPI:

- Users `users.schema.json`
    ```json showLineNumbers copy filename="user.schema.json"
    {
        "name": "User",
        "length": 15,
        "content": [
            {
                "key": "username",
                "type": "username",
                "prefix": "@"
            },
            {
                "key": "status",
                "type": "options",
                "options": [
                    true,
                    false
                ]
            }
        ]
    }
    ```
