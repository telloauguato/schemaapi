# Options

The `options` type returns one or more values, depending on the number of options passed to it, as specified by the `length` attribute.

## Properties

| Name | Type | Description | Required |
| ------------- | --------- | -------- | --------- |
| `type` | `string` | Defines the type of data that will be returned. Must be assigned as `options`. | Yes |
| `length` | `number` | Sets the default length when less than `0`. This is optional. | No |
| `options` | `array` | Defines the data to be randomly drawn to be returned as an array if the `length` is greater than 1, otherwise it returns the drawn value. | Yes |
| `order` | `boolean` | When `true` sorts the values ​​in order. By default it is `false`. | No |
