# Name

O tipo `name` no SchemaAPI é projetado para gerar nomes fictícios realistas, úteis em cenários de desenvolvimento e teste onde dados de identificação pessoal são necessários, mas não devem corresponder a indivíduos reais.

import { Callout } from 'nextra/components'
 
<Callout type="info" emoji="ℹ️">
  A **Propriedade Herdada** desse tipo é a [String](/inherited/string)
</Callout>

## Propriedades

| Nome | Tipo  | Descrição | Requerida |
| ------------- | -------- | -------------- | --------- |
| `type`        | `string` | Tipo de dado que será retornado. Deve ser atribuído como `name`. | Sim |
| `join` | `boolean` | 	Define se o resultado deve ser retornado como string (`true`) ou objeto (`false`). O valor padrão é `true`. | Não |
 
<Callout type="info" emoji="ℹ️">
  Quando se utiliza atributos da [**Propriedade Herdada String**](/inherited/string) e o atributo `join` está configurado como `true`, esses atributos retornam dentro do corpo do objeto de retorno.
</Callout>
