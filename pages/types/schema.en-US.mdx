# Schema

The `schema` type is designed to create secondary schemas and returns them as objects, allowing the user to create more complex data hierarchies.

## Properties

| Name | Type | Description | Required |
| ------------- | --------- | -------- | --------- |
| `type` | `string` | Defines the type of data that will be returned. Must be assigned as `schema`. | Yes |
| `length` | `number` | Sets the default length when less than `0`. This is optional. | No |
| `schema` | `object` | Defines the data types to be returned as a schema object. | Yes |
