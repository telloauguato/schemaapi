# Number

O tipo `number` é projetado para representar valores numéricos, úteis em diversos cenários de modelagem de dados.

## Propriedades

| Nome | Tipo  | Descrição | Requerida |
| ------------- | -------- | -------------- | --------- |
| `type`        | `string` | Tipo de dado que será retornado. Deve ser atribuído como `number`. | Sim |
