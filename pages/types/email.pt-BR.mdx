# Email

O tipo `email` é projetado para representar endereços de email válidos. Ele é útil em cenários onde é necessário armazenar ou manipular dados de contato, garantindo que os valores inseridos sigam o formato padrão de endereços de email.

## Propriedades

| Nome | Tipo  | Descrição | Requerida |
| ------------- | -------- | -------------- | --------- |
| `type`        | `string` | Tipo de dado que será retornado. Deve ser atribuído como `email`. | Sim |
| `domain`        | `string` | Escreva um doinio para ser o dominio ficticio dos dados ou um array de dominios | Sim |
