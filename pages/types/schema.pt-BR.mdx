# Schema

O tipo `schema` é projetado para criar esquemas secundários e os retorna como objetos, permitindo que o usuário crie hierarquias de dados mais complexas.

## Propriedades

| Nome          | Tipo      | Descrição   | Requerida |
| ------------- | --------- | -------- | --------- |
| `type`        | `string`  | Define o tipo de dado que será retornado. Deve ser atribuído como `schema`. | Sim       |
| `length`      | `number`  | Define o comprimento padrão quando menor que `0`. Este é opcional.          | Não       |
| `schema`      | `object`  | Define os tipos de dados a serem retornados como um objeto de esquema.      | Sim       |
