# Options

O tipo `options` retorna um ou mais valores, dependendo do número de opções passadas para ele, conforme especificado pelo atributo `length`.

## Propriedades

| Nome          | Tipo      | Descrição   | Requerida |
| ------------- | --------- | -------- | --------- |
| `type`        | `string`  | Define o tipo de dado que será retornado. Deve ser atribuído como `options`. | Sim       |
| `length`      | `number`  | Define o comprimento padrão quando menor que `0`. Este é opcional.  | Não       |
| `options`      | `array`  | Define os dados a serem sorteados aleatoriamente para serem retornados como um array caso o `length` for maior que 1, se não retorna o valor sorteado.      | Sim       |
| `order`      | `boolean`  | Quando `true` ordena os valores por ordem. Por padrão é `false`.     | Não       |
