# Propriedades Herdadas

As propriedades herdadas referem-se às propriedades que são automaticamente atribuídas a um tipo baseado em sua definição ou comportamento padrão. Em essência, essas propriedades são inerentes ao tipo e são aplicadas independentemente de como ele é utilizado.

Aqui está a lista das propriedades herdadas disponíveis:

import { Cards, Card } from "nextra/components";
import { CaseSensitive, Hash, Braces, Brackets } from "lucide-react";

<Cards>
  <Card icon={<CaseSensitive />} title="String" href="/inherited/string" />
  <Card icon={<Hash />} title="Number" href="/inherited/number" />
  <Card icon={<Braces />} title="Object" href="/inherited/object" />
  <Card icon={<Brackets />} title="List" href="/inherited/list" />
</Cards>
