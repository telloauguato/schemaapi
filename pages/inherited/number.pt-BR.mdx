# Number

A **Propriedade Herdada Number** adiciona atributos opcionais ao tipo. Esses atributos são `decimail`, `max`, `min`, `increment` e `decrement`.

```json filename="Tipo number"
{   
    "type": "number",
    "decimails": 2,
    "max": 100,
    "min": 50,
    "increment": 10,
    "decrement": 0.69
}
```

- **`decimal`**: Define se o número pode conter casas decimais.
- **`max`**: Especifica o valor máximo permitido para o número.
- **`min`**: Especifica o valor mínimo permitido para o número.
- **`increment`**: Define o valor a ser incrementado quando uma operação de incremento é aplicada.
- **`decrement`**: Define o valor a ser decrementado quando uma operação de decremento é aplicada.
