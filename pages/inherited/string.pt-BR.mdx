# String

A **Propriedade Herdada String** adiciona atributos opcionais ao tipo. Esses atributos são `suffix`, `prefix` e `format`.

```json filename="Tipo email"
{   
    "type": "email",
    "domain": "active.inc",
    "suffix": ".internal",
    "prefix": "admin.",
    "format": "lowercase"
}
```

- **`suffix`**: Adiciona um texto que será colocado no final da string.
- **`prefix`**: Adiciona um texto que será colocado no início da string.
- **`format`**: Define o formato da string, podendo ser em maiúsculas (`uppercase`) ou minúsculas (`lowercase`).
