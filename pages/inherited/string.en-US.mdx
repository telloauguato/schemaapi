# String

The **String Inherited Property** adds optional attributes to the type. These attributes are `suffix`, `prefix` and `format`.

```json filename="Email type"
{
    "type": "email",
    "domain": "active.inc",
    "suffix": ".internal",
    "prefix": "admin.",
    "format": "lowercase"
}
```

- **`suffix`**: Adds text that will be placed at the end of the string.
- **`prefix`**: Adds text that will be placed at the beginning of the string.
- **`format`**: Defines the format of the string, which can be uppercase (`uppercase`) or lowercase (`lowercase`).

The types that receive properties inherited from the string are:
