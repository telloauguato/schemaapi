#Number

The **Number Inherited Property** adds optional attributes to the type. These attributes are `decimail`, `max`, `min`, `increment` and `decrement`.

```json filename="Number type"
{
    "type": "number",
    "decimals": 2,
    "max": 100,
    "min": 50,
    "increment": 10,
    "decrement": 0.69
}
```

- **`decimal`**: Defines whether the number can contain decimal places.
- **`max`**: Specifies the maximum value allowed for the number.
- **`min`**: Specifies the minimum value allowed for the number.
- **`increment`**: Defines the value to be incremented when an increment operation is applied.
- **`decrement`**: Defines the value to be decremented when a decrement operation is applied.
