## Overview

The `number` type is designed to represent numeric values, useful in various data modeling scenarios.

## Parameters

| Name          | Type      | Description                          |
|---------------|-----------|--------------------------------------|
| `type`        | `string`  | Type of data that will be returned. Must be assigned as `number`. |
