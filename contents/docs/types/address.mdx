## Overview

The `uuid` type is designed to generate a complete fictitious address, including street, city, state and zip code.

## Parameters

| Name          | Type      | Description                          |
|---------------|-----------|--------------------------------------|
| `type`        | `string`  | Type of data that will be returned. Must be assigned as `address`. |
