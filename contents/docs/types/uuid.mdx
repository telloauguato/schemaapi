## Overview

The `uuid` type is designed to generate universally unique identifier (UUID).

## Parameters

| Name          | Type      | Description                          |
|---------------|-----------|--------------------------------------|
| `type`        | `string`  | Type of data that will be returned. Must be assigned as `uuid`. |
